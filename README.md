# Nextjs CORS
A test of nextjs with cross origin resource sharing.

## Usage
Used with [Apixer: API Tester](https://apixer.vercel.app/)

## To run this project:
1. Clone this repository `git clone https://github.com/khianvictorycalderon/nextjs-cors.git`
2. Run `npm install`
3. Run `npm run dev`
